webpack的使用：
　  １.初始化工程：npm init
    2.安装webpack:npm install webpack -D
    3.安装webpack-cli:npm install webpack-cli -D
    4.安装的东西都在node_modules文件夹下，node_modules文件夹下有webpack文件夹
    5.命令行查看版本./node_modules/.bin/webpack -v(ubuntu系统下)
    6.调用webpack指令需要到bin文件夹下，调用太过麻烦，我们可以安装npx，使用npx运行初始安装的webpack包的webpack文件
    ７．使用 npx　webpack -v可以直接使用webpack命令，不用添加路径了
